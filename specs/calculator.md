# Calculator Spec

## Feature: Basic arithmetic operations

### Scenario: Add two numbers
- Given two integers `a` and `b`
- When `add(a, b)` is called
- Then it returns `a + b`

### Scenario: Subtract two numbers
- Given two integers `a` and `b`
- When `subtract(a, b)` is called
- Then it returns `a - b`
